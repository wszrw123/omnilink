设置原则

                
这个公开项目优先保持边界清晰，而不是预埋复杂扩展能力。

              

            

            
    
              
  • 当前仓库保持单通道模型，不引入 provider 切换。
    • 
              
  • 所有可变状态都写到用户数据目录，不写进应用包。
    • 
              
  • 跨平台差异保留在主进程，不暴露到 renderer。
    • 
              
  • 当前 mock runtime 只是公开版占位，后续替换成真实 worker。
    •
